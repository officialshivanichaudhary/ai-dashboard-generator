## вҡҷпёҸ Setup & Run Locally

1. Clone the repository:

```bash
git clone https://github.com/officialshivanichaudhary/ai-dashboard-generator.git
```

2. Navigate to project:

```bash
cd ai-dashboard
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:3000
```
